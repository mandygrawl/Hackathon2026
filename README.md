# Hackathon2026
Red Devil Hacks 2026 Submission

Project Name: VR Find the Key Game

Group: Mandy Grawl; Liam Glackin, Lukas Hamm, and Addy Deysu as alpha testers

How to play: download the .apk file, plug in meta quest headset (with developer mode enabled) to pc, use SideQuest to load the .apk file on the headset, play!

Why? I made a VR game to push the limits of my knowledge in C# coding and physics simulation, with a fun premise of needing to find the key in each room to progress to the next while being able to interact with nearly every item in the room. It was mostly a project to exercise my 3D modeling skills by custom-making every item in the game, and many people in the hackathon had a fun time playtesting to find bugs when taking breaks. 

I did not use AI at all for this project. All assets and code are my own or built in to Unity 6.
